## 公式分析

y=0时，A(x,y)=0,

x=0且y!=0时，A(x,y)=2y,

x!=0且y=1时，A(x,y)=2,

否则，A(x,y)=A(x-1,A(x,y-1))

## 推导

f(n) = A(0, n) = 2n

g(n) = A(1, n) = A(0, A(1, n-1)) = 2g(n-1), 且g(1) = A(1, 1) = 2, 那么g(n) = 2^n

h(n) = A(2, n) = A(1, A(2, n-1)) = 2^A(2, n-1) = 2^h(n-1), 且h(1) = A(2, 1) = 2, 那么h(n)=2^2^..^2 (n个2的阶乘)

## 计算的例子

A(1,10) = 1024

A(2,4) = 65536
